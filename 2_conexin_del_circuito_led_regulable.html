2. Conexión del circuito: Led regulable

En la siguiente imagen se muestra las conexiones que se deben realizar para implementar la electrónica del presente proyecto. Además, se detalla las  conexiones a continuación:

  1. El pin 0 del microbit conéctalo con la terminal A0 del módulo del sensor de luz.
  2. El pin 1 del microbit conéctalo con la terminal positiva del módulo LED.
  3. El pin 3V del microbit conéctalo con la terminal VCC del módulo del sensor de luz.
  4. El pin GND del microbit conéctalo con la terminal negativa del módulo LED.
  5. La terminal GND del módulo LED conéctalo con la terminal GND del módulo del sensor de luz.
